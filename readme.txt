== Description ==

This plugin adds a button to the CKEditor plugin's Link tool that allows you to select a WordPress page.

Once installed,just click on the "Link" tool in the toolbar, click on the "Wordpress Page" button, and select the page you wish to link to.

This plugin requires that you install the [CKEditor plugin](http://wordpress.ckeditor.com/).
